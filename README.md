# Infill

infill.hackclub.com

Welcome to infill! In-*fill* the gaps you see in 3D printing with your own printer design. This is *your* chance to join other hack clubbers in designing your own 3D printer.

## Website

The website is built with [Astro](https://astro.build/) and [Starlight](https://starlight.astro.build/).

### Development

```bash
cd website
npm install
npm run dev
```

### Building

```bash
cd website
npm run build
```

## How it works

1. **Design a 3D printer** - Come up with a design and start building!
2. **Submit a BOM** - Get a $350 grant to buy parts
3. **Build and tune** - Make it print a perfect benchy
4. **Get flown to RMRRF** - Join us in Colorado, April 18-19, 2026

Join [#infill on the Hack Club Slack](https://hackclub.com/slack) to get started!
